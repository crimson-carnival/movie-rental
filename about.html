This site is our project for the subject 'XML Programming'

For the project, we are design a website using Hyper Text Markup Language, Cascading Style Sheets, Javascript, PHP (PHP Hypertext Preprocessor) along with XML for storing the data at the backend. A DTD and Schema were created first, in order to create a storage schema for data in XML and later, the backend was integrated with the same, to generate the required XML document. The website is named "Haebe-Flix" because "Haebe-Haebe" is Dann Kudo's favourite phrase in gibberish.

The project is a part of our subject, XML Programming, in the fifth semester of our course, Bachelor of Technology in Computer Science and Engineering with specialisation in Open Source and Open Standards, in School of Computer Science, University of Petroleum and Energy Studies.

In the bottom section of the page, one might find links to masquerade.96.lt and catipsum.com in the help centre. That's because catipsum.com is a really fun page that generates the traditional lorem isum text, but adds a "cat" element to it, and masquearde.96.lt is Crimson Carnival's blogging site for bloggers to blog anonymously and you'll be helping him because he desperately needs members. catipsum.com is featured in the "Join us" section too because we want you to join us in generating those funny paragraphs. Vision & Mission

Haebe-Flix is a movie rental website that showcases movies to a user. When a user clicks on a movie, they are taken to the rentals page where they can check if the movie is available to rent, has already been rented, or is in the "coming soon" phase. If the movie is available, the user can rent the movie by clicking on the "rent" button. Otherwise, they have to wait for it to be returned.

The rentals can only be marked on this website as the actual copies of the DVD's need to be collected and returned to Crimson Carnival and Dann Kudo's offices: C-115, Charley Block, UPES Energy Acres Campus Hostel, Bidholi, via Prem Nagar, Dehradun, Uttarakhand, India, Asia, Earth, Solar System, Milky Way, Universe - 248007 (we apologise if the address is a bit of a stretch).
